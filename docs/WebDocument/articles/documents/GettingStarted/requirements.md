動作環境
==========

開発のために必要な環境
----------
### Windows
- Windows 7, 8, 8.1, 10

### macOS
- macOS 10.11+

----------
### Windows
- Windows 7, 8, 8.1, 10
- シェーダーモデル 4.0 対応のグラフィックスカード

### macOS
- macOS 10.11+
- OpenGL 3.2+

### iOS
- iOS 7+
- OpenGL ES 3.0+

### Android
- Android 5.0+
- OpenGL ES 3.0+

> [!Warning]
> 0.7.0 時点ではエミュレータ上で動作確認を行っています。

### Web
- Chrome, Firefox
